04 - Findall  
The findAll() method in Spring Data JPA is provided by the JpaRepository interface,  
retrieving all the records from the database associated with the entity.  
Example:  
@SpringBootApplication  
public class SpringDataJpaExApplication {  
public static void main(String[] args) {  
ApplicationContext context=  
SpringApplication.run(SpringDataJpaExApplication.class, args);  
Student s1= context.getBean(Student.class);  
Student s2=context.getBean(Student.class);  
Student s3=context.getBean(Student.class);  
StudentRepo repo=context.getBean(StudentRepo.class);  
System.out.println(repo.findAll());  
}
}
