12 - React UI for Delete And Update  
In this, we implement update and delete functionality for job posts on the React  
frontend, connecting with backend endpoints.  
Backend API Endpoint for Update: @PutMapping("jobPost") is used to handle  
update requests for job posts.-  
● Edit Component Setup:  
○ Fetches the job post data by ID from the backend using axios.get and  
populates the form with the data retrieved.  
○ useLocation(): Captures the current post ID from the location state.  
○ useNavigate(): Allows redirection after successful submission if  
needed.  
○ useState(): Manages form data and stores the current post ID.
