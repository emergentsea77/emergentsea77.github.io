Running an Embedded Tomcat Server  
Tomcat Configuration:  
● Tomcat Instance: new Tomcat() initializes an embedded Tomcat server.  
● Persistent Server: tomcat.getServer().await() ensures the server keeps  
running and listens for requests.  
HTTPServlet:  
service() Method: Handles incoming HTTP requests and generates appropriate  
responses.  
Tomcat tomcat = new Tomcat();  
tomcat.start();

Ensure the server keeps running:  
tomcat.getServer().await();
