.jar (Java ARchive): A package of Java classes, libraries, and metadata bundled  
together for use in applications.  
.war (Web Application Archive): A specialized archive format for web

● Standalone Tomcat: Requires manual deployment of .war files in the

● Embedded Tomcat: Integrated into your application via Maven  
dependencies, making it easier to manage.
