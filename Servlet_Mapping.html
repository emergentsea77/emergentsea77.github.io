Servlet Mapping  
Servlet Mapping:  
● Annotation-based Mapping: Using @WebServlet("/hello") for automatic  
URL mapping in external Tomcat.  
● Manual Mapping: In embedded Tomcat, use methods like  
addServletMappingDecoded() to associate URLs with specific servlets.  
Manual Mapping for Embedded Tomcat:  
Context context = tomcat.addContext("", null);  
tomcat.addServlet("HelloServlet", new HelloServlet());  
context.addServletMappingDecoded("/hello", "HelloServlet");  
tomcat.start();  
tomcat.getServer().await();  
Output:  
Custom Port Configuration:  
tomcat.setPort(8081) changes the default port from 8080.
