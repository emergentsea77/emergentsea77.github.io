Setting Prefix and Suffix  
View Resolver:  
● Defines how Spring locates view files using prefix (/views/) and suffix  
(.jsp).  
Now we have viewresolver to resolve and identify where the views file is and with  
which name it is associated.  
@RequestMapping("add")  
public String add(@RequestParam("num1") int num1, @RequestParam("num2") int  
num2, Model model) {  
int result = num1 + num2;  
model.addAttribute(attributeName: "result", result);  
return "result";  
}
Configure application.properties  
spring.mvc.view.prefix=/views/  
spring.mvc.view.suffix=.jsp
